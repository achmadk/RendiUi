# Rendi-Ui

Memudahkan anda dalam membuat component user interface reactjs.

## Instalasi

Anda dapat menginstal komponen ini menggunakan npm.

```bash
npm install rendi-ui

Penggunaan
Di bawah ini adalah cara penggunaan dasar komponen ini dalam proyek React Anda:

import React from 'react';
import {Button} from 'rendi-ui';

function App() {
  return (
    <div>
      <Button 
      variant="contained" //outlined
      color="primary"    //secondary,success,warning,disabled
      > this is button</Button>
    </div>
  );
}

export default App;
